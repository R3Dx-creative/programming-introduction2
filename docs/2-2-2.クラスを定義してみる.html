2-2-2.クラスを定義してみる

分数クラス

整数の分子と分母を持つBunsuクラスを定義します(変に英語を使うと読みづらいのでローマ字にします(普段はあまりローマ字を使わないでね))。

class Bunsu
  def initialize(bunshi, bunbo)
    @bunshi = bunshi
    @bunbo = bunbo
  end
end

classの横にクラスの名前を付け、情報(ここでは分子と分母)をオブジェクトの内部に保持するためのinitialize(初期化)というメソッドを定義します。Rubyでは内部の情報を示す変数には「@」を先頭につける必要があります。(@の付いた内部の変数のことをインスタンス変数と呼びます)

クラスを定義をした後、以下のようにnewメソッドを呼び出すことで分数を表現したオブジェクトを取得できます。(クラスからオブジェクトを作り出すことをインスタンス化(実体化)と呼びます。)

class Bunsu
  def initialize(bunshi, bunbo)
    @bunshi = bunshi
    @bunbo = bunbo
  end
end

x = Bunsu.new(8, 10)

値を取得

Rubyではオブジェクトの内部に情報を持つことにしているので中の変数を見るには情報を見るための処理を書く必要があります。

class Bunsu
  attr_reader :bunshi, :bunbo

  def initialize(bunshi, bunbo)
    @bunshi = bunshi
    @bunbo = bunbo
  end
end

x = Bunsu.new(8, 10)
puts x.bunshi # xのbunshi
puts x.bunbo  # xのbunbo

attr_readerに知りたい変数のキーワードを指定することで内部の情報をメソッドを介して取得できるようになります。

メソッドを定義

ここでは単純な分数の掛け算だけ定義します。 余裕があったら足し算も定義してみてください。

分数の掛け算は単純に分子は分子で掛けて、分母は分母で掛ければOKです。

class Bunsu
  attr_reader :bunshi, :bunbo

  def initialize(bunshi, bunbo)
    @bunshi = bunshi
    @bunbo = bunbo
  end

  def mul(other)
    new_bunshi = @bunshi * other.bunshi
    new_bunbo = @bunbo * other.bunbo

    Bunsu(new_bunshi, new_bunbo)
  end
end

計算してみましょう

class Bunsu
  attr_reader :bunshi, :bunbo

  def initialize(bunshi, bunbo)
    @bunshi = bunshi
    @bunbo = bunbo
  end

  def mul(other)
    new_bunshi = @bunshi * other.bunshi
    new_bunbo = @bunbo * other.bunbo

    Bunsu(new_bunshi, new_bunbo)
  end
end

x = Bunsu(8, 10)
y = Bunsu(3, 16)

a = x.mul(y)
puts a.bunshi
puts a.bunbo
